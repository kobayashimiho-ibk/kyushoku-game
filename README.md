# 🍱 きゅうしょくじゅんびゲーム

特別支援学校 小学部１年生向けの給食準備ゲームです。

## ▶ あそびかた

`index.html` をブラウザで開くだけで遊べます。インストール不要！

## 📋 ゲーム内容

| # | ステップ | ミニゲーム |
|---|---------|-----------|
| 1 | 手洗い | タップ連打（泡アニメ） |
| 2 | クイズ：何を使う？ | 3択クイズ |
| 3 | エプロンを着る | タップで装着 |
| 4 | 三角巾をつける | タップで装着 |
| 5 | クイズ：なぜつける？ | 3択クイズ |
| 6 | マスクをつける | タップで装着 |
| 7 | クイズ：なぜマスク？ | 3択クイズ |
| 8 | タオルを濡らす | 蛇口タップ（水アニメ） |
| 9 | クイズ：次は何する？ | 3択クイズ |
| 10 | タオルをしぼる | しぼるボタン連打 |
| 11 | どれがどれかな？ | マッチングクイズ |
| 12 | 順番に並べる | シーケンスゲーム |
| 13 | クイズ：並べる順は？ | 3択クイズ |
| 14 | 座って待つ | 椅子タップアニメ |
| 15 | クイズ：準備できたら？ | 3択クイズ |

## ✨ 特徴

- 全文ひらがな
- 大きな文字・大きなボタン
- 正解音・不正解音（Web Audio API）
- ステップクリア時に✨パーティクル演出
- 全クリア時に🎆花火アニメ＋ファンファーレ
- 進捗バー＋⭐スター表示

## 🏫 対象

特別支援学校 小学部１年生
